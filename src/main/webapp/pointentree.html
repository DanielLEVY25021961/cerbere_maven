pointentree.html sous WebContent

Cette page nommée pointentree.html est écrite en XHTML 1.0 Strict avec un encodage UTF-8.

Elle est située directement sous la racine (contexte) du serveur ('WebContent' sous ECLIPSE), donc à la position 'http://localhost:8080/cerbere/pointentree.html' si le serveur est 'localhost:8080' et le contexte 'cerbere'.

Cette page est servie même en tapant dans l'URL du navigateur 'http://localhost:8080/cerbere/pointentree.html' puisqu'elle est directement sous le contexte (WebContent de Eclipse).

ATTENTION :

Cette page ne peut être directement servie en tapant 'http://localhost:8080/cerbere/pointentree.html' que si l'utilisateur s'est déjà loggé sous Cerbere.

En effet, le filtre Cerbere dans le web.xml intercepte TOUTES les requêtes, sauf les URL "ouvertes" : 

				<filter-mapping>
    					<filter-name>
    						<url-pattern>/*</url-pattern>
    					</filter-name>
    				</filter-mapping>

Et "contexte/pointentree.html" ne fait PAS partie des URL ouvertes.

  • vers l'accueil = /WEB-INF/jsp/application/accueil.jsp via l'action accueil.do (http://localhost:8080/cerbere/accueil.do)
  • vers pointentree.jsp sous le contexte (http://localhost:8080/cerbere/pointentree.jsp)
